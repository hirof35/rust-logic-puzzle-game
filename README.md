# Logic Matrix Pro (Rust)

C++ (Siv3D) で作成された論理マトリクス（ロジックパズル）の解答支援・自動推論ツールを、Rust + `macroquad` に移植した作品です。
<img width="1601" height="942" alt="スクリーンショット 2026-05-17 153501" src="https://github.com/user-attachments/assets/e5501a64-ec9a-4e9f-af39-d1e042d535ae" />

## 特徴
- **自動推論エンジン**: 1つの条件（○や×）を確定させると、三段論法（ワーシャル–フロイド法ベース）を用いて他のマスを自動的に割り出します。
- **推論エフェクト**: 自動推論によって変化したマスに、視覚的な波紋エフェクトが発生します。
- **履歴管理**: 50手まで遡れる Undo / Redo 機能を搭載（`Ctrl+Z` / `Ctrl+Y` にも対応）。
- **セーブ/ロード**: 軽量バイナリシリアライザ `bincode` を使用し、経過時間と盤面データを保存します。
- **テーマ切り替え**: ダークモードとライトモードをリアルタイムに切り替え可能です。

## 実行方法

```bash
cargo run --release
操作方法
左クリック: マスを 「○」 -> 「×」 -> 「空欄」 の順に切り替えます。

Undo / Redo: 画面のボタン、または Ctrl + Z / Ctrl + Y

セーブ: 画面右下の「セーブ」ボタンを押すと、save.dat が生成されます。
